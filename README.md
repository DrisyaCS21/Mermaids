Overview
Mermaids is an underwater survival game developed using Python and Pygame. In this game, the player controls a mermaid who is being chased by a shark. The main objective is to avoid the shark and survive as long as possible by controlling the movement of the mermaid.

Features
*Player-controlled mermaid movement
*Shark chasing mechanism
*Interactive and simple gameplay
*Underwater-themed game environment

Technologies Used
*Python
*Pygame

How to Run the Game
1.Install Python on your system.
2.Install the required library using:
 |_pip install pygame
3.Run the game file:
 |_python main.py

Objective:
The goal of the game is to control the mermaid carefully and prevent the shark from catching her. The longer the player survives, the higher the score or gameplay duration.
